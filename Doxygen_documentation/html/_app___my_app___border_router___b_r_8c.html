void responseHandler ( const uint8_t *  payload,
uint8_t  payload_size 
)

Response handler used to check if border router got correct response for "Turn on the light" request send to leaf node.
If so, GPIO corresponding with LED diode is setting in HIGH state.
Similary, if border router got response for "Turn off the light", corresponding with LED diode GPIO is setting in LOW state.

Parameters
[in]ipv6_payloadresponse from leaf node.
[in]payload_sizesize of response from leaf node.
